title: 'Create and Configure Prompt'

---

Follow these steps to create and configure a prompt in Hyperleap:

## Prerequisites

Before creating prompts, ensure you have:
- Access to [Studio](https://studio.hyperleapai.com)
- Credits in your account

<Steps>
  <Step title="Get Started">
There are two ways to create a prompt

    <Tabs>
      <Tab title="Start from Scratch">
        Start creating your Prompt from Prompt Playground
        
        <Frame>
          <img 
            src="/images/prompts/prompt-pg.png" 
            alt="Creating a blank prompt" 
            width="800"
          />
        </Frame>
      </Tab>
      <Tab title="Use Template">
        Choose pre-configured Prompt from Prompt Library
        
        <Frame>
          <img 
            src="/images/prompts/prompt-library.png" 
            alt="Selecting a prompt from prompt library" 
            width="800"
          />
        </Frame>
        <Frame>
          <img 
            src="/images/prompts/import-prompt.png" 
            alt="importing a prompt" 
            width="800"
          />
        </Frame>
      </Tab>
    </Tabs>
  </Step>

  <Step title="Configure Replacements">
    Define or update your Prompt input. You can include dynamic variables (e.g. `{{variable_name}}`) that will be replaced with actual values when the Prompt is used.
    
    - Use `{{variable_name}}` syntax for replacements
    <Note>
    To learn more about replacements, check out the [Replacements](/guides/replacements) guide.
    </Note>

    <Frame>
      <img 
        src="/images/prompts/add-replacements.png" 
        alt="Adding dynamic variables" 
        width="800"
      />
    </Frame>

  </Step>

  <Step title="Configure Settings">
    <AccordionGroup>
      <Accordion title="Set AI Model" icon="microchip">
        Choose the AI model that best fits your use case:
        <Frame>
          <img 
            src="/images/prompts/set-model.png" 
            alt="Setting AI model" 
            width="800"
          />
        </Frame>
      </Accordion>

      <Accordion title="Attach Sources" icon="database">
        Connect your data sources to ground AI responses:
        - Add documents, website URLs, or text content
        <Note>
        To learn more about sources, check out the [Sources](/guides/sources) guide.
        </Note>
        <Frame>
          <img 
            src="/images/prompts/sources.png" 
            alt="Attaching data sources" 
            width="800"
          />
        </Frame>
      </Accordion>

      <Accordion title="Adjust Configurations" icon="sliders">
        Fine-tune your prompt parameters:
        <Frame>
          <img 
            src="/images/prompts/adjust-configuration.png" 
            alt="Adjusting configurations" 
            width="800"
          />
        </Frame>
      </Accordion>

      <Accordion title="Access Template Library" icon="file-import">
        Access our template library at any point during configuration:
        - View available templates by category
        - Preview template behaviors
        - Apply template settings to your current prompt
        
        <Frame>
          <img 
            src="/images/prompts/import-prompts.png" 
            alt="Importing prompts" 
            width="800"
          />
        </Frame>
      </Accordion>
      <Note>
        This is an alternative way to access templates if you didn't start with one. Your existing configurations will be overridden.
      </Note>
    </AccordionGroup>
  </Step>

  <Step title="Test Prompt">
    Test your prompt configuration in the playground:
    - Enter dynamic values
    - View generated responses
    
    <Frame>
      <img 
        src="/images/prompts/generate-response.png" 
        alt="Generating response" 
        width="800"
      />
    </Frame>
    <Frame>
      <img 
        src="/images/prompts/enter-inputs.png" 
        alt="Entering values" 
        width="800"
      />
    </Frame>
  </Step>
  
  <Step title="Compare Models">
    Compare the performance of different AI models:
    - View response quality
    - Adjust configurations
    - Analyze response time
    <Frame>
      <img 
        src="/images/prompts/compare-models.png" 
        alt="Comparing models" 
        width="800"
      />
    </Frame>
  </Step>

  <Step title="Save and Activate">
    Finalize your prompt configuration:
    - Save as draft to continue editing and share with your team later
    - Publish to make it available to your team
    
    <Frame>
      <img 
        src="/images/prompts/save-prompt.png" 
        alt="Saving and activating prompt" 
        width="800"
      />
    </Frame>
  </Step>
</Steps>


<Note>
  Users on the Free plan can create up to 10 prompts. Upgrade your plan to create more prompts and access additional features.
</Note>
