## General Questions

<AccordionGroup>
  <Accordion title="What can I build with Hyperleap AI?">
    You can build AI-powered applications using:
    - Prompts for structured AI interactions
    - Personas for conversational experiences
    - Tools for no-code AI workflows
    - Assistants for team support
    - Chatbots for website interactions
  </Accordion>

  <Accordion title="How do I get started?">
    1. Sign up at [studio.hyperleapai.com](https://studio.hyperleapai.com)
    2. Create your first teamspace
    3. Choose a feature to start with (prompts, personas, tools, etc.)
    4. Follow our quickstart guides
  </Accordion>

  <Accordion title="What are the usage limits?">
    Limits vary by plan:
    - Free: 10 prompts, personas, or tools
    - Growth: Increased limits and features
    - Business: Custom limits and dedicated support

    refer to [pricing page](https://hyperleapai.com/pricing) for more details
  </Accordion>
</AccordionGroup>

## API Integration

<AccordionGroup>
  <Accordion title="How do I authenticate API requests?">
    All API requests require an API key:
    1. Get your key from Studio dashboard
    2. Add header: `x-hl-api-key: YOUR_API_KEY`
    3. Keep your key secure
  </Accordion>

  <Accordion title="How do I handle streaming responses?">
    We support two streaming methods:
    - SSE streaming: For real-time chat responses
    - HTTP/2 streaming: For prompt runs
    
    Check our [API guides](/api-ref/introduction) for implementation details.
  </Accordion>
</AccordionGroup>

## Features & Capabilities

<AccordionGroup>
  <Accordion title="What's the difference between prompts and personas?">
    - **Prompts**: Single-shot AI interactions
    - **Personas**: Ongoing conversations with custom AI personality
  </Accordion>

  <Accordion title="Can I customize the AI models?">
    Yes, you can:
    - Choose from multiple LLM providers
    - Adjust temperature and other parameters
    - Fine-tune responses with custom configurations
  </Accordion>

  <Accordion title="How does content moderation work?">
    Built-in moderation:
    - Automatic content screening
    - Safety filters for inputs/outputs
    - Customizable moderation rules
  </Accordion>
</AccordionGroup>
